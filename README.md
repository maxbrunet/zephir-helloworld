# zephir-helloworld

Zephir Helloworld from the [Zephir documentation](https://docs.zephir-lang.com/en/latest/tutorial).

## Requirements

* [zephir](https://github.com/phalcon/zephir)

## Build

```
cd helloworld
zephir build
```

Test it!:

```
$ php -r 'echo Helloworld\Greeting::say(), "\n";'
hello world!
```
